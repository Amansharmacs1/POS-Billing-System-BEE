# 🛒 POS Billing System

A simple **Point of Sale (POS) Billing System** built using **HTML, CSS, and JavaScript** as part of my Backend Engineering coursework. This project demonstrates DOM manipulation, event handling, and dynamic bill generation using core web technologies.

## 🚀 Features

* Add products to the bill
* Update product quantity
* Remove products from the bill
* Automatic price calculation
* Dynamic bill generation
* Responsive and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📁 Project Structure

POS-Billing-System/
│── index.html      # Main application structure
│── styles.css      # Styling
│── script.js       # Application logic
└── notes.md        # Project notes


## 📚 Concepts Practiced
